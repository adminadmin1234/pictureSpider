## 说明
#### 这是一个练习的nodejs爬虫，可以简单的爬取网页中的图片，并且保存到本地。
#### 由于我选择的这两个网站对爬虫没有限制，所以可以一直爬取，但是我希望大家不要过度爬取损害别人的利益，我们要以学习为主。

## 运行操作
#### 环境
- nodejs(本人用的是v8.11.2)
- 安装依赖
```bash
npm install
```

#### 执行
- 您可以选择index.js或者index1.js这是对两个不同网站dom的爬取操作
```bash
node index
```
或
```bash
node index1
```
执行成功后，图片文件会下载到imgae文件夹中。
